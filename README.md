# Dottore Agronomo Michele Ghezzi — sito statico minimo

Questo pacchetto contiene un sito statico minimale, pronto per GitHub Pages, per il repository:

`micheleghezzi/dottore-agronomo`

## Obiettivo del sito

Il sito presenta Michele Ghezzi come Dottore Agronomo specializzato in consulenza tecnica per coltivazioni con mezzi biologici e naturali, con particolare attenzione a:

- piccoli frutti e fragole;
- coltivazioni fuori suolo;
- piani di concimazione biologica;
- difesa da funghi e insetti con principi attivi biologici;
- canapa, orticoltura, verde ornamentale, cerealicoltura e frutticoltura;
- supporto a produttori e tecnici sull'uso corretto dei prodotti biologici.

## Come caricare su GitHub

1. Aprire il repository `micheleghezzi/dottore-agronomo`.
2. Cliccare **Add file** → **Upload files**.
3. Trascinare tutti i file e le cartelle contenuti in questo pacchetto.
4. Scrivere come messaggio di commit: `Add initial static website`.
5. Cliccare **Commit changes**.

## Come attivare GitHub Pages

1. Nel repository, aprire **Settings**.
2. Nel menu laterale, cliccare **Pages**.
3. In **Build and deployment**, scegliere **Deploy from a branch**.
4. Selezionare:
   - Branch: `main`
   - Folder: `/ (root)`
5. Cliccare **Save**.

Quando GitHub Pages termina la pubblicazione, il sito dovrebbe essere disponibile a:

`https://micheleghezzi.github.io/dottore-agronomo/`

# Image and rights policy for this website

## Safe by default

Use:
- First-party headshot provided by Michele.
- First-party farm/field/berry photos provided by Michele.
- Public pages as external links.

## Do not use without permission

Do not copy, download, crop, or reuse:
- ItalianBerry article images
- Macfrut/Berry Area photos, PDF slide visuals, or logos
- Cosmogarden event photos
- CLT / Fondazione Fojanini images or logos
- LinkedIn or Facebook profile photos
- Book cover images from retailers or publisher pages

## Safer alternatives

- Use text-only cards with external links.
- Use simple abstract botanical illustrations created specifically for the site.
- Use first-party photos from Michele.
- Use stock images only if properly licensed and documented.

## If using logos

Use organization logos only if:
1. The organization’s brand guidelines allow it, or
2. Michele has written permission, or
3. The site uses text links instead of logos.

## Asset metadata to store

For every image added to the site, store:

```yaml
filename:
source:
owner:
permission_status:
date_received:
allowed_use:
credit_required:
notes:
```
